Welcome to Forge

Here is where you will begin the process of forging decks. Adding cards can be done from any device. But "Forge a deck" should be done from a computer. This will create ".apkg" files with the recently added cards within them. When you try and open them, this should trigger anki to open and import them. If not, just open anki, click import files and import the recently downloaded deck. For more info please see the "How it works" in the navigation bar above.



Common questions


I've added a card, but I can't see it in my deck in "Forge a deck"?


TLDR; This should normally take around 15 seconds per card sent. But could take a little longer if there are a lot of people sending quotes at once.

When you add a card, it is sent to our server. The server then needs to translate it, assign native and learnt languages. Then it has to run language processing to make sure we find relevant images, search for the image and get the synthesised speech. The image, audio and quote is then uploaded to a cloud ready to be made into a Anki card when you want.



I lost the download of a deck I forged?


All of the decks ".apkg" files you forge will be available in "My forged decks". They are all time stamped and can be downloaded at anytime.
