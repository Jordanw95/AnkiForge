How AnkiForge works in 4 steps...

Step 1

Sign up and subscribe

Sign up from the link in the top right. Once you've made an acccount, you will be prompted to subcribe through Stripe Payment Systems. Follow the steps. You'll then receive an email that we've received your payment and you'll be ready to begin. You can cancel your subscription at anytime.

Step 2

Make a deck

Now you need to create a deck template so AnkiForge know's how you'd like your cards. You need a name to recognise each deck on AnkiForge. You also need a name for the deck in Anki. Important, if you want the cards to be added to a deck that you are already using - you need to make sure the name in the Anki Deck field matches exactly with the name of your deck in Anki. Otherwise, just name the deck whatever you'd like it to be called in Anki. Select the language you'd like to learn in this deck. Finally, you need to select the style for the cards, you can preview the available styles here.

Step 3

Add cards

Now we are ready to begin adding cards. This is the part that will begin to use your monthly credit. The deduction from your credit is dependent on the number of characters that you send. With an average card length of 50 characters, you can make around 400 cards each month. From the Forge section of the website, go to add a card. Then simply select the deck you want to send cards to, enter the quote and click send! This can be done from mobile or your desktop. There is also a desktop application available, that can be downloaded here. This will allow you to send quotes using the keyboard shortcut "ctrl/cmd + c" followed by "ctrl/cmd + x".

  • As countless Anki users will tell you, it is best to use anki cards with complete sentences. This helps add context to a word and understand the gramma. For me personally, this has definitely been true. However, if you want to use Ankiforge for individual words it will still work well.
  • I would always recommend sending quotes in your learnt language, using example sentences from dictionarys, books or TV. This way you get more natural and native sounding cards and audio. It's also much easier for you to spot mistakes from the translator in your native language and edit the card.
  • My workflow when I want to add a word is to go to a dictionary, get an example sentence and send it to ankiforge.
  • If you are going to add quotes in English, I recommend adding some context to the sentence so that the translator can pick the correct word. For example: sending "We met yesterday" to spanish gives "Nos conocimos ayer". This implies you met someone for the first time. However, if you send "We met for a drink" it will give "Nos encontramos para tomar una copa", which is more accurate for meeting a friend.

Step 4

Forge your deck

Finally, forge your deck. Simply select the deck on the left pane and you will see all the cards you've sent that are ready to be made into a deck. This should always been done from a desktop. You will only see cards here that have not yet been made into ".apkg" files. You can forge into decks as often as you like (I prefer to do it every few days after building up a few cards). Once forged, you will be prompted to download the deck, you then just need to click the download and it should open and import to Anki. If that doesn't work, just open up Anki and click import at the bottom, and select the file you just made. If you want to download your previously forged decks, just go to My forged decks. Here you'll see all your forged ".apkg" files for that deck.
