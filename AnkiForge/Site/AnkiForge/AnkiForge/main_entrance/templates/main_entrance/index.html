Why did I make this?

When I first started learning Spanish, I found Anki indispensable. For me, it was the best way to learn a language. However, I found the process of making good quality cards, from sentence mining, time consuming. So I found a way to automate it. I found it so useful that I thought other people would want to use it too.

Why isn't this free?

It costs me money. All of services used (image search, translation, Azure Nueral Voices) for AnkiForge are premium. I wanted somthing reliable. This combined server costs, database costs and workers to make the media and cards, really add up. Finally, I want this to continue functioning and improving, with your feedback! I have lots of plans to improve it. Paying the subscription fee is what will make this happen.

Is it perfect?

Absolutely not. The service relies heavily on translation services. This means it's only as good as the translation service it's using (Google Cloud Translation for now, but I plan to test out DeepL). However, this doesn't mean it's not extremely useful. The real benefit of this service is that adding a word or phrase to your Anki deck (and ultimately your vocabulary) is now a split-second decision, that can be done from anywhere. There's also tips to make it work best for you.
