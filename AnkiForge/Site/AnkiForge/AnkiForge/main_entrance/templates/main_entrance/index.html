Why did I make this?

When I first started learning Spanish, I found Anki in-dispensible. It was a sure-fire way to commit vocab to memory. However, I found the process of making good quality cards, from sentence mining, time consuming. So I made a python script to do it for me. I found it so useful that I thought other people would want to use it too. Fast forward 6 months and here we are.

Why isn't this free?

The primary reason - it costs me money. All of the APIs used for the service (image search, translation, azure nueral voices) are premium. Sure, there are ways to do this for free - but I wanted somthing reliable. This combined with the hosting cost of the server, workers to make the media and cards, and databases to store everything, really add up. Finally, I've put a lot of work into making this work and I want it continue to work! I also have lots of plans to improve it. Paying the subscription fee is what will make this happen.

Is it perfect?

Absolutely not. The service relies heavily on translation APIs. This means it's only as good as the translation service it's using (Google Cloud Translation for now, I'm weighing up the benefits of deepL). However, this doesn't mean it's not extremely useful. The real benefit of this service is that adding a word or phrase to your Anki deck (and ultimately your vocabulary) is now a split-second decision, that can be done from anywhere. There's also a host of tips to make it work best for you.
